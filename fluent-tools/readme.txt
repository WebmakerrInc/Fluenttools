FluentTools unifies FluentAuth security hardening with FluentSMTP email delivery into a single WordPress plugin.

== Description ==

FluentTools bundles everything you need to secure your WordPress login and reliably deliver transactional emails in one plugin. Protect your site with FluentAuth's login security, 2FA, magic login links, and social login providers while simultaneously routing email through FluentSMTP's powerful connection manager.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/fluent-tools` directory, or install directly through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure authentication and email delivery from the FluentTools dashboards in the WordPress admin area.

== Frequently Asked Questions ==

= Where can I configure the security settings? =

Navigate to `wp-admin > FluentAuth` to manage authentication security options.

= Where can I configure the email settings? =

Navigate to `wp-admin > Settings > Fluent SMTP` to manage SMTP connections and email logs.
